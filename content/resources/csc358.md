### What is CSC358?

CSC358 is a course that focuses on the system programming of networks. This
course will teach students the basics of computer networks and network protocols

### Topics in CSC358:

- Network Hardware
- Network Software
- Routing
- Addressing
- Congestion Control
- Reliable Data Transfer
- Socket Programming
